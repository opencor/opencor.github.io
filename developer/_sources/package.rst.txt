.. _package:

=========
 Package
=========

Packaging of OpenCOR can only be done from the command line, using |packageScript|_\[|packageBatch|_].
However, as for :ref:`building OpenCOR <buildTestAndRunFromTheCommandLine>`, it can be done in parallel.
