.. _plugins_editing_editingView:

====================
 EditingView plugin
====================

The EditingView plugin is a non-selectable plugin that provides core editing view facilities.

.. _plugins_editing_editingView_uriScheme:

URI scheme
----------

The following commands are supported by the EditingView plugin:

- ``EditingView.setCursorPosition``: set the cursor position of the current editor.

  - Example: |editingViewSetCursorPosition|_.

.. |editingViewSetCursorPosition| replace:: ``opencor://EditingView.setCursorPosition/3|5``
